# RA de CPS

L’avaluació de les competències personal i socials es portarà a terme dins de cada Resultat d’Aprenentatge com a un Criteri d’Avaluació de cada mòdul. Per fer-ho, el professorat de manera conjunta utilitzarà aquesta rúbrica:

## COMPETÈNCIES PERSONALS I SOCIALS

| Competència | Descripció i Indicadors | Gens (1) | Poc (4) | Força (7) | Molt (10) | Pes |
| :--- | :--- | :---: | :---: | :---: | :---: | :---: |
| **Emprenedora** | S'adapta de forma autònoma a les dificultats amb l'objectiu de trobar una solució efectiva.  
• Proposa i du a terme solucions segons les necessitats

• Té iniciativa per buscar solucions  
• Pregunta constructivament i amb curiositat | ☐ | ☐ | ☐ | ☐ | 25.00 % |
| **Digital** | Domina la utilització de les eines TIC i utilitza la més adequada en cada cas.  
• Respecta el format i estructura demanats

• Implementa la solució usant l'eina correcta | ☐ | ☐ | ☐ | ☐ | 25.00 % |

• Mostra una escolta activa
• Fa el que toca fer en cada moment | ☐ | ☐ | ☐ | ☐ | 25.00 % |

• Té cura del material i les instal·lacions
• Mostra una actitud professional (puntual, compromès) | ☐ | ☐ | ☐ | ☐ | 25.00 % |
